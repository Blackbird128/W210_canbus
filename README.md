# W210_canbus

Dedicated hardware interface for communication with the CAN-B network of a Mercedes-Benz W210 (W202 or W208)
The board is based on the MCP2515 CAN controller and the TJA1055 fault-tolerant transceiver, compatible with ISO 11898-3 low-speed CAN networks.
